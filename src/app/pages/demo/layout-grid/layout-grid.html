Explicit Row Start

u-row-start-* pins an item to a specific row, leaving any skipped rows empty. There is no row equivalent of u-col-end-last: rows are implicit, so a negative row line has nothing to count back from. Use u-row-* to span downwards.

Explicit Row Tracks

By default a grid only has the rows its content creates. u-grid-rows-* declares real row tracks on the container, so u-row-start-* lands on a known row. Responsive variants (u-grid-rows-lg-3) work like the column ones.

Implicit Row Sizing

u-auto-rows-* sizes the rows a grid creates on its own - auto, min, max, or fr for equal heights. It applies without declaring any row template.
